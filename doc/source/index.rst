`flowws-analysis <https://github.com/klarh/flowws-analysis>`_ is an
in-development library to create workflows for analysis of
simulations.

`flowws-analysis` only contains stages for workflows; to see how to
use the stages, consult the `flowws documentation
<https://flowws.readthedocs.io>`_.

Installation
============

Install `flowws-analysis` from source on github::

  pip install git+https://github.com/klarh/flowws-analysis.git#egg=flowws-analysis
